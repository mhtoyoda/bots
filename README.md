# icaptor-framework

## pre-requisitos
Start Platform Server
-  ```java -jar target/icaptor-cloud-0.0.1-SNAPSHOT.jar``` 

start client
- ```java -Dspring.config.location=file:$HOME/.icaptor-out-file/icaptor-platform.properties -jar target/agent-icaptor-0.0.1-SNAPSHOT.jar```

start server
- ```java -jar target/icaptor-cloud-0.0.1-SNAPSHOT.jar```
